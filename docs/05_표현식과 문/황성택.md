### 5.1 값

__[예제 05-02]__

```javascript
// 변수에는 10 + 20이 평가되어 생성된 숫자 값 30이 할당된다.
var sum = 10 + 20;
```

- 값은 식이 평가되어 생성된 결과를 말한다.<br>
평가❓ 식을 해석해서 값을 생성하거나 참조하는 것
- 모든 값은 데이터 타입을 가지며, 메모리에 2진수(=비트)의 나열로 저장된다.
